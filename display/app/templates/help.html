Using the Robot

1. Initial Settings

Go to Innitial Settings and fill in the required fields:
  • Experiment Name
  • Person Responsible
  • Experiment Description
  • Number of Cycles
  • Thermomixer Time
  • LN2 Time
  • Number of Samples
After filling in the fields, click on "Save Settings" to save the experiment settings.

2. Loading Samples

DO NOT force the robot to move. It should move easily, if not contact help.
  • Place the samples in the holder
  • Tighten the top gentally with the screw

3. Starting the Experiment

After the samples are loaded, go to Start Page. Find the experiment setting in the dropdown menu or enter the experiment id in the search field and select it. If you have just created the experiment, it should be the last one in the list. Confirm the settings and click on "Start Experiment". The robot will start the experiment and you can monitor its progress on the Status Page.

4. Monitoring the Experiment

Go to Status Page to monitor the progress of the experiment. The page will automatically refresh every 700 milliseconds to show the latest status. You can see the current cycle, last action, and time since the last action.

5. Completing the Experiment

If the experiment is completed, you can add post-experiment information in the provided textbox and click "Finish Experiment" to save it. You can also download the experiment data as a JSON file.

6. View Experiments

Go to Completed Experiments Page to view all completed experiments. click on the experiment name to view its details, including the settings and robot actions. Json with all the details of the experiment can be downloaded by clicking on "Download JSON".
